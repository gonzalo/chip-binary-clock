# CHIP Binary clock
This code is used in combinations with the CHIP computer by NTC and some extra hardware to create a physical binary clock. Feel free to recommend and extend this project.

Created and mantained by Gonzalo Cao Cabeza de Vaca. Please send any feedback or comments to gonzalo.cao(at)gmail.com

### Version
0.1
### Tech
In order to construct this project you'll need some extras
* [CHIP](https://www.getchip.com/pages/chip) computer by NTC
* [CHIP_IO python library](https://github.com/xtacocorex/CHIP_IO)
* 18 x 3.3v leds
