# CHIP Binary clock
This code is used in combinations with the CHIP computer by NTC and some extra hardware to create a physicall binary clock. Feel free to recommend and extend this project.

Created and mantained by Gonzalo Cao Cabeza de Vaca. Please send any feedback or comments to gonzalo.cao(at)gmail.com

0.1
### Tech
In order to construct this project you'll need some extras
* [https://github.com/xtacocorex/CHIP_IO][CHIP_IO python library]
* 18 x 3.3v leds.
